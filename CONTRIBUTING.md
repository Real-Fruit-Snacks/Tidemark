# Contributing to Tidemark

## Development Setup
1. Clone into your Obsidian vault's `.obsidian/plugins/tidemark/` directory
2. Run `npm install`
3. Run `npm run dev` for hot-reload development
4. Enable the plugin in Obsidian Settings > Community Plugins

## Code Style
- TypeScript with strict mode
- Follow Obsidian plugin API conventions

## Pull Request Process
1. Fork and create a feature branch
2. Test in Obsidian
3. Submit PR with description

## Commit Messages
Use conventional commits: `feat:`, `fix:`, `docs:`

No AI co-author signatures in commits.
