# Code of Conduct

## Our Pledge
We are committed to providing a friendly, safe, and welcoming environment for all contributors.

## Our Standards
**Expected:** Welcoming language, respect for differing viewpoints, graceful acceptance of criticism.
**Unacceptable:** Harassment, trolling, personal attacks, publishing private information.

## Enforcement
Report unacceptable behavior via GitHub issues.
