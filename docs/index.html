// capabilities

Write once. Fill everywhere.

Define variables in YAML frontmatter, reference them with {{}} syntax, then copy or replace with a single keystroke. Every feature works with markdown files.

✎

Variable Replacement

Replace {{variables}} with YAML frontmatter values. Copy to clipboard or permanently replace in-document. Supports nested properties, arrays, and defaults.

📋

Quick Copy

Copy current line, selection, or entire document with variables replaced. Paste commands straight into your terminal with values already filled in.

🎨

Syntax Highlighting

Color-coded variables: green for existing, orange for defaults, red for missing. See variable status at a glance without running anything.

🔍

Variable List

Searchable modal shows every variable grouped by status. Edit values, navigate to locations, and identify missing variables in one view.

🔗

Nested Properties

Access nested YAML with dot notation: {{server.ip}}, {{creds.user}}. Array indexing with {{items[0]}}. Toggle nesting on or off.

📄

File Renaming

Replace variables in filenames too. Rename files based on frontmatter values with sanitization and collision detection built in.

⚙

Configurable

Custom delimiters, default values, case-insensitive matching, array join separators, notification levels. Every aspect is tunable.

⌨

Keyboard Shortcuts

All commands available via Command Palette. Configure your own hotkeys in Obsidian Settings for fast access to every common operation.

// quickstart

Productive in 30 seconds

Install the plugin, open a markdown note with frontmatter, and start using variables immediately.

notes/target.md
---
IPAddress: 10.10.10.1
hostname: target
username: admin
---
# Enumeration
nmap -A -p- {{IPAddress}}
# Use Command Palette to copy with values filled
ssh {{username}}@{{hostname}}

1. Install — Search "Tidemark" in Obsidian Settings > Community Plugins > Browse, and click Install.

2. Write — Create a markdown file with YAML frontmatter between --- fences. Reference values with {{variable}} syntax.

3. Use — Open Command Palette (Ctrl/Cmd+P), type "Tidemark", and select "Copy current line (replaced)". Paste directly into your terminal.

4. Replace — Use "Replace all in document" from Command Palette to permanently replace all variables when you're done templating.
