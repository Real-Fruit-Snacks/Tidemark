Capabilities

Variable substitution, solved.

Define variables in YAML frontmatter, reference them anywhere in your document, then copy or replace with a single command.

Mustache-Style Syntax

Use {{variable}}, {{variable:default}}, and {{nested.path}} to reference frontmatter values.

Live Syntax Highlighting

CodeMirror decorations color-code variables by status (Green = set, Orange = fallback, Red = missing).

9 Palette Commands

Copy or permanently replace variables by line, selection, or entire document with dedicated commands.

Nested Properties & Arrays

Supports dot-notation for nested YAML structures and automatic array joining with customizable separators.

Workflow

Define. Reference. Substitute.

1

Define Variables

Add YAML frontmatter to any note with your desired variables.

2

Reference Inline

Reference them in the document body using {{variable}} syntax and see live syntax highlights.

3

Copy or Replace

Open the Command Palette (`Ctrl/Cmd+P`) and search for "Tidemark" to permanently replace variables or copy to clipboard.

Get Started

Install in under a minute.

Recommended

Community Plugins

  1. Open Obsidian Settings and navigate to Community Plugins.
  2. Browse and search for Tidemark.
  3. Install and Enable the plugin.

Manual

  1. Download main.js, manifest.json, and styles.css from the latest release.
  2. Create .obsidian/plugins/tidemark/ inside your vault.
  3. Place the files inside that folder and reload Obsidian.
