Features

Variable substitution that works the way you think. Define once in frontmatter, use everywhere in your document.

Variable Replacement

Replace {{variables}} with YAML frontmatter values. Copy to clipboard or permanently replace in-document. Supports nested properties, arrays, and defaults.

Quick Copy

Copy current line, selection, or entire document with variables replaced to clipboard. Paste commands with values already filled in.

Syntax Highlighting

Color-coded variables: green for existing, orange for defaults, red for missing. See status at a glance.

Nested Properties

Dot notation for nested YAML: {{server.ip}}, {{credentials.user}}. Array indexing with {{items[0]}}.

Variable List

See every variable grouped by status, edit values, and navigate to locations. Right-click any variable to set its value directly.

Default Values

Fallback values with {{port:443}} syntax. Missing variables show [MISSING] or preserve the original — your choice.

Custom Delimiters

Change {{"{{"}} / {{"}}"}} to any characters. Custom default separator. Case-insensitive matching option.

Array Support

YAML arrays automatically joined with configurable separator. {{ports}} becomes 22, 80, 443.

File Rename

Replace variables in document body and filename simultaneously. Create templated files that name themselves.

Capabilities

Full variable substitution engine running inside Obsidian. Works on desktop and mobile.

Architecture

Clean separation between parsing, replacement, and UI. TypeScript throughout with Obsidian's plugin API.

1

Frontmatter Parser

Extracts YAML frontmatter from the active document. Handles nested objects, arrays, and scalar values. Dot-notation resolution for deep property access.

2

Variable Replacer

Core replacement engine. Pattern matching with configurable delimiters. Default value extraction. Case-insensitive option. Array join with custom separator.

3

Decoration Provider

CodeMirror 6 decorations for syntax highlighting. Three colors by variable status. Real-time updates as you edit frontmatter values.

4

Command System

Nine commands registered with Obsidian's Command Palette. Copy, replace, rename, list, and set operations. Configurable hotkeys.

Getting Started

Install from Community Plugins or build from source.

install.sh 
# Build from source
$ git clone https://github.com/Real-Fruit-Snacks/Tidemark.git
$ cd Tidemark

# Install and build
$ npm install
$ npm run build

# Copy to vault
$ cp main.js manifest.json styles.css \
    /path/to/vault/.obsidian/plugins/tidemark/

Plugin ready — enable in Obsidian settings
example.md 
---
IPAddress: 10.10.10.1
username: admin
ports: 22,80,443
---

# Enumeration

nmap -A -p- {{IPAddress}}
ssh {{username}}@{{IPAddress}}
gobuster dir -u http://{{IPAddress}}

# Copy line → paste in terminal
$ nmap -A -p- 10.10.10.1
