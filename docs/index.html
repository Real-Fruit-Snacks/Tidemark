FEATURES

Template any note

Define variables in YAML frontmatter, reference them with {{variable}} syntax, then copy or replace with a single command.

Variable Replacement

Define variables in YAML frontmatter and reference them with {{variable}} syntax anywhere in your document. Copy to clipboard or permanently replace in-document with a single command. Supports nested dot-notation and array joins.

target_ip: 10.10.10.1
nmap -sV -sC {{target_ip}}
=> nmap -sV -sC 10.10.10.1
# Nested: {{server.ip}} => 10.10.10.1
9
Commands

Default Values

Provide fallback values with the : separator. {{port:443}} uses 443 if port is not defined in frontmatter. Never leave a missing variable blank.

Nested Properties

Access nested YAML structures with dot notation. Array values automatically joined. {{server.ip}}, {{credentials.user}}, {{ports}} all resolve correctly.

Syntax Highlighting

Variables color-coded in the editor: green for resolved, orange for default-only, red for missing. Colors adapt to light and dark themes automatically.

ARCHITECTURE

TypeScript + esbuild

Clean separation between YAML parsing, variable resolution, and CodeMirror decorations. Single-file output for Obsidian.

1

Frontmatter Parser

YAML frontmatter extraction with support for nested objects, arrays, and dot-notation access. Resolves {{server.ip}} through arbitrary nesting depth.

2

Replacement Engine

Core variable resolution with default value fallback ({{port:443}}), case-insensitive matching option, and configurable delimiters. Array values joined with separator.

3

Decoration Provider

CodeMirror decorations for real-time syntax highlighting. Three status colors: green (resolved), orange (default only), red (missing). Custom hex overrides in settings.

4

Command Handlers

Nine commands registered via Obsidian API. Copy, replace, rename, and list operations. Context menu integration for quick value setting directly from the editor.

GETTING STARTED

Install or build

Install from Obsidian Community Plugins, or build from source with Node.js 18+.

build.sh
$ git clone https://github.com/Real-Fruit-Snacks/Tidemark.git
$ cd Tidemark

$ npm install
$ npm run build

# Copy to your vault
$ cp main.js manifest.json styles.css \
    /path/to/vault/.obsidian/plugins/tidemark/
usage.md
---
target_ip: 10.10.10.50
hostname: target.htb
wordlist: /usr/share/wordlists/common.txt
---

nmap -sV -sC {{target_ip}}
gobuster dir -u http://{{target_ip}} \
  -w {{wordlist}}
echo "{{target_ip}} {{hostname}}" >> /etc/hosts
